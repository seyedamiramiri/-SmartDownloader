## 🔹 گزینه‌های خروجی

حالت (mode):
- normal: دریافت عادی
- zip: فشرده‌سازی

بخش‌بندی (split):
- 0: بدون تکه کردن
- 500: تکه‌های ۵۰۰ مگی
